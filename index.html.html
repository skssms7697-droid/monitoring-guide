유선 연락 우선순위

긴급도별 담당자 연락 순서와 대응 시간 기준입니다.

🔴

긴급

서버 다운, 전체 결제 장애. 즉시 유선.

🟠

높음

일부 결제 오류, 심각 버그. 15분 내 유선.

🔵

보통

UI 깨짐, 간헐적 오류. 1시간 내 유선.

🟢

낮음

오타, 번역 오류. 메신저만.

이슈 유형별 처리 방법

주요 이슈 유형과 대응 매뉴얼입니다.

🖥

서버 장애

서버 정보 기록 후 긴급 보고. 교차 확인.

💳

결제 장애

무조건 긴급. UID/시각/금액 기록.

🐛

인게임 버그

재현 경로 기록. 간헐적이면 조건 파악.

🔒

보안/어뷰징

증거 확보 후 보고. 빠르게 전달.

📢

커뮤니티 여론

링크+반응 요약 전달. 직접 대응 금지.

🔄

업데이트/점검

정상 동작 전수 확인. 체크리스트 점검.
